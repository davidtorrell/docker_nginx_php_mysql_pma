# DOCKER
- Nginx
- PHP 7.3-fpm
- MySQL
- PHPMyAdmin

## :rocket: Quickstart 
- Install and launch Docker  
- `cp .env.dev .env`  
- `docker-compose up`

| Service      | Path                    |
| ------------ | ----------------------- |
| Website      | `http://localhost:8080` | 
| PhpMyAdmin   | `http://localhost:8081` |
| Logs         | `log/`                  |
